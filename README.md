# WinTool-MacOS
## About
App for aligning windows on macOS
## How to run this app?
First of all you need to install the compiler for swift.
### macOS:
#### Installing the command-line-tools:
```zsh
xcode-select --install
```
#### Compiling the project:
```zsh
swift build
```
#### Running the project
```zsh
./.build/debug/build
```
